# raman_spectroscopy_nuclear_IT_hack


График показывает химический состав образцов. По сути, это "молекулярный отпечаток пальца" — каждый пик рассказывает о том, какие вещества присутствуют в образце.


По горизонтали (ось X) отложен Рамановский сдвиг — это характеристика энергии колебаний молекул. Измеряется в обратных сантиметрах (см⁻¹). На нашем графике диапазон от 950 до 2000 см⁻¹ охватывает важные колебания биологических молекул.

По вертикали (ось Y) показано количество отсчетов (Counts). Это интенсивность сигнала — сколько фотонов зарегистрировал детектор на каждой частоте.


Когда лазер освещает образец, фотоны взаимодействуют с молекулами. Небольшая часть фотонов рассеивается, и детектор их подсчитывает. Чем больше молекул определенного типа в образце, тем больше фотонов рассеивается и тем выше значение Counts.

Высокие Counts (например, 90 000) означают сильный сигнал и большое количество молекул. Низкие Counts (20 000) — слабый сигнал и мало молекул.


Каждый пик на графике соответствует колебаниям определенных химических связей в молекулах. Положение пика (его координата по оси X) говорит о том, какая это молекула, а высота пика (значение по оси Y) показывает концентрацию этой молекулы в образце.


Чем выше пик, тем больше соответствующих молекул в образце.


Сравнивая линии, мы видим различия в химическом составе между группами. Если одна линия выше другой в области определенного пика, значит в этой группе больше соответствующих молекул.
